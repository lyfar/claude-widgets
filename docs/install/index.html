Install on iPhone

Copy the widget source, paste it into Scriptable. ~30 seconds.

Before you start:
  • Scriptable installed on this iPhone
  • Tailscale running on both iPhone and Mac
  • Mac-side installer has been run (curl … | bash) and its server is up

Enter your Mac's Tailscale IP above.

In Scriptable, do this once

  1. Tap + in the top-right corner to create a new script.
  2. Paste the code (it's already on your clipboard).
  3. Tap the script's name at the top → rename to Claude Code → Done.
  4. Tap ▶︎ Play once to preview. You should see live rate-limit numbers.
  5. Long-press the home screen → + → Scriptable → pick Small / Medium / Large → select Claude Code.
View widget source (if clipboard copy didn't take)

Long-press the textarea → Select All → Copy.

One script, every size

The Claude Code script powers every widget size. Add as many widgets to the home screen as you like — each can be Small, Medium, or Large, plus Lock-screen circular / rectangular / inline. The script detects which placement is calling it and renders accordingly.

Light / dark follows your system setting automatically. To force a specific theme on a single widget, long-press it → Edit Widget → set Parameter to light or dark.
