Install on iPhone

One tap sends the universal widget script straight into Scriptable.

Before you tap install:
  • Scriptable installed on this iPhone
  • Tailscale running on both iPhone and Mac
  • Mac-side installer has been run (curl … | bash) and server is up

Opens Scriptable with a tiny Install Claude Code script. Tap Save, then Run (▶︎) once — it pulls the full widget from this site and saves it as Claude Code. Delete the installer after.

Or paste the full widget manually

New Scriptable script → paste → name it Claude Code → save. No internet needed after this.

What you get from one install

A single script named Claude Code powers every widget size. Add as many widgets to the home screen as you like — each can be Small, Medium, or Large, plus Lock-screen circular / rectangular / inline. The script detects which placement is calling it and renders accordingly.

Light / dark follows your system setting automatically. To force a specific theme on a single widget, long-press it → Edit Widget → set Parameter to light or dark.
